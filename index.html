Welcome to Glip.

Glip is a pure PHP library to read and commit into a git repository.

The original code of Glip has been developed by Patrik Fimml (@patrikf) and the actual version contains changes made by many contributors some years ago.

Since 2021, it is maintained by Laurent Jouanneau.

If you want to contribute, feel free to fork the project and submit pull requests.
